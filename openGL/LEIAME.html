Lista de Conteúdo

Este pacote contém os seguintes programas:

  • basico - programa esqueleto
    Este programa mostra a estrutura básica de um programa que usa GLUT e OpenGL. Não tem comandos para desenhar nada.
  • basicoDesenho - programa com desenho
    Este programa desenha um triângulo.
  • basicoTeclado - programa que trata o teclado
    Este programa mostra como associar ações ao toque de teclas.
  • basicoTransformacao - programa que apresenta transformações
    Este programa gira um triangulo ao toque de uma tecla.
  • basicoTransformacaoDB - programa que apresenta transformações
    Este programa gira um triangulo ao toque de uma tecla, usando double buffer. Note como não se nota falhas no desenho durante a animação.
  • camera - mostra como usar o teclado para produzir o efeito de movimentação do observador.
  • pilhaMatrizes - mostra como usar as matrizes GL_PROJECTION e GL_MODELVIEW em pilha para evitar o efeito acumulativo das transformações.
